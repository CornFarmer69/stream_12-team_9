DEEP LEARNING

Our chosen topic is Deep Learning, deep learning is Artificial intelligence which is made to imitate the processing and working of the human brain in order to make decisions. Deep learning is a collection of machines capable of learning from the network about data from unlabelled and unstructured sources. Deep Learning, machine learning and artificial intelligence are basically a subset of each other. While both fall under artificial intelligence, deep learning is usually what is behind the most human-like artificial intelligence. Machine learning requires user input e.g., when detecting facial features, and is highly dependent on the accuracy of the features it is fed, whereas deep learning tries to learn the high-level features itself from data. In order for the detection to be precise the system must have a vast number of models to base the detection off and give an accurate result. Deep learning has its advantage and its disadvantages.
